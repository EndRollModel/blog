Hexo - 幫Fluid加上CC0著作

目前因為 Fluid 主題尚未增加 CC0 1.0 的標註

但在Fluid的[issue/1053]有提出該問題 但目前還未新增CC0相關的內容

原本是想說無著作權則不標註作者欄位與著作權宣告

但這個宣告模組太漂亮了 我想把它擺在下面展示 XD

最後只好手動新增並且做個紀錄了

修改以下內容

  1. 修改檔案
  • 檔案路徑 : theme/fluid/languages/zh-TW.yml
    copyright 中新增一行
    CC0: 'CC0 - 公眾領域'
  1. 修改檔案
  • 檔案路徑 : theme/fluid/layout/_partails/post/copyright
    行數：50 
    <%- license %>
    替換成以下內容 
    <% if (license.toUpperCase() === 'CC0') { %>
    <a class="print-no-link" target="_blank" href="https://creativecommons.org/public-domain/cc0/">
       <span class="hint--right hint--rounded" aria-label="<%- __('post.copyright.CC0') %>">
           <img class="iconfont icon-cc-zero" style="width: 16px; height: 16px;" src="<%= config.root%>img/cc-zero.svg" alt=""/>
       </span>
    </a>
<% } else { %>
    <%- license %>
<% } %>

  1. 使用該選擇器選擇CC0的版權 存下該圖 將圖放入/Img並將檔名命名為cc-zero.svg

  2. 修改_config.fluid.yml中的copyright / license值為CC0就可以使用了
